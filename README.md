# WHO-analysis-# Exploratory Data Analysis on NHANES Dataset

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **NHANES (National Health and Nutrition Examination Survey)** dataset. The goal is to understand the dataset, clean the data, identify patterns, and generate meaningful insights through statistical analysis and visualizations.

The notebook demonstrates a complete EDA workflow, including data inspection, preprocessing, feature analysis, and graphical representation of important findings.

---
